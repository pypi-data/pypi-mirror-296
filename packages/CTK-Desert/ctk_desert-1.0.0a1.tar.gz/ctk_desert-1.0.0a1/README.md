## Usage

1. Create an assets folder in the project's directory:

    ```bash
    cd project_dir/
    mkdir Assets  # You can name this folder anything
    ```
2. Script to use:

    ```python
    from CTK_Desert import Desert
    import os

    assets_dir = os.path.join("Assets")
    Desert(assets_dir=assets_dir)
    ```
